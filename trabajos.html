Chatfox.es

Es una aplicación para Firefox OS. Es un chat abierto vinculado a twitter que recibe notificaciones cada vez que un usuario envía un mensaje a través del chat o desde algunas de las cuentas de twitter vinculadas a éste. Ha sido desarrollada para pruebas de rendimiento y pionera en el uso de notificaciones push en Firefox OS. Está hecha con node.js, javascript y mongodb. Es accesible desde móvil, tableta o computadora con navegador Firefox.
